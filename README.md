# ESLint-Configured Node.js Project

This project showcases the setup and configuration of ESLint in a Node.js project.

## Project Setup

1. **Clone Repository**: Clone this repository to your local machine using the following command:

   ```bash
   git clone https://github.com/el-riber/web4.git
   ```

2. **Navigate to Project Directory**: Navigate to the project directory in your terminal:

   ```bash
   cd eslint-node-project
   ```

## ESLint Configuration

- **ESLint Configuration File**: The ESLint configuration file `.eslintrc.js` is provided in the root directory of the project.
- **Extends ESLint Rules**: The configuration extends recommended rules from ESLint, TypeScript, and React plugins.
- **Customized Rules**: Rules for semicolon usage and quotes have been adjusted to match the project's coding style preferences.

## Running ESLint

To lint JavaScript or TypeScript files using ESLint, use the following command:

```bash
npx eslint myfile.js
```

Replace `myfile.js` with the path to your JavaScript or TypeScript file.

## Customizing ESLint Rules

To further customize ESLint rules, edit the `.eslintrc.js` file in the root directory of the project. Refer to the ESLint documentation for available rules and their configurations.
